# matsuo_APR

## 内容
- 学生の講義向けにテストを作成し，Semantic errorに対する自動プログラム修正とsyntax errorに対する自動プログラム修正を併用した実験を行う．

## ディレクトリ構成
- test
  - 授業用テスト 
- script
  - 各種実験で利用したスクリプト群が入ってます
  - 一定の汎用性が保障されているものはREADMEを用意してます
- Tex
  - 会議名のディレクトリにその会議の原稿が入ってます
- slide
  - 会議名のディレクトリにその会議のスライドが入ってます
  - 其の他雑多なスライドがあるかも
- other
  - その他作成した雑多なファイルがあります
  - 基本ここの中のものは他者に公開しない前提で適当に作ってるので，ご容赦ください
