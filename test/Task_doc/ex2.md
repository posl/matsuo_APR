## 課題1
04-Loop.pdfのベクトルの内積を計算するプログラムを書き換えて，キーボードからの入力を読み込むようにしてください．

### 入力形式
```
(ベクトル1の値1)
(ベクトル1の値2)
(ベクトル1の値3)
(ベクトル2の値1)
(ベクトル2の値2)
(ベクトル2の値3)
```
入力は**少数**でも受け付けられるようにしてください．
浮動小数点の範囲("float"型の範囲)で受け付けられるようにしてください．

### 出力形式
```
(内積の値)
```

出力桁数は**浮動小数点数型の出力範囲**(**"%f"で出力できる範囲**)までで行ってください．

### 具体例

*入力例1*
```
1
2
3
4
5
6
```
*出力例1*
```
32.000000
```
<div style="page-break-before:always"></div>

*入力例2*
```
1.2
-2.7
-4.5
-2.0
-1.6
-2.3
```
*出力例2*
```
12.270000
```

### 注意事項
ベクトルは3次元ベクトルという前提でプログラムを作成して構いません．

### コピペ用プログラム
```main.c
#include <stdio.h>

int main()
{
    int i;
    double ip = 0.0;
    double u[]={1.9, 2.8, 3.7};
    double v[]={4.6, 5.5, 6,4};

    for (i=0; i<3; i++){
        ip = ip + u[i]*v[i];
    }
    printf("%f\n", ip);
    
    return 0;
}

```
