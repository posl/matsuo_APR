## 課題2
03-Function_If.pdfのスライド37ページを参考に，摂氏と華氏を変換するプログラムを作成してください．
摂氏と華氏の変換は次の式で行われます．

```
F = 9/5×C + 32
F：華氏の温度
C：摂氏の温度
```

### 入力形式
入力は以下の形式で与えられます．
```
(摂氏の温度)
```
入力は**少数**でも受け付けられるようにしてください．
浮動小数点の範囲("float"型の範囲)で受け付けられるようにしてください．

*具体例*
```
-15.4
```

### 出力形式
出力は以下のような形式で行ってください．
```
(華氏の温度)

```

出力桁数は**浮動小数点数型の出力範囲**(**"%f"で出力できる範囲**)までで行ってください．
```
81.139999

```


### 注意事項
必ず摂氏と華氏の変換を行うユーザー定義関数(convC2F)を定義してください．
テストに通るかに加えて，ユーザー定義関数を使っているかについても評価します．

出力は最後の値の後に必ず改行を行ってください．
改行を行わないとテストに通りません．

## 課題3
03-Function_If.pdfのスライド40ページを参考に2次方程式の解を求めるプログラムを作成しなさい．

### 入力形式
入力は以下の形式で与えられます．
```
(aの値)
(bの値)
(cの値)
```
入力は**少数**でも受け付けられるようにしてください．
浮動小数点の範囲("float"型の範囲)で受け付けられるようにしてください．

*具体例*
```
1
0.5
-0.5
```

### 出力形式
出力は解の形によって場合分けしてください．

異なる2つの実数解の時
```
(実数解1)
(実数解2)

```

重解の場合
```
(実数解1)

```

虚数解の場合
```
(実部)+(虚部)i
(実部)-(虚部)i

```

出力桁数は**浮動小数点数型の出力範囲**(**"%f"で出力できる範囲**)までで行ってください．

異なる2つの実数解の時
```
-2.000000
-3.000000

```

重解の場合
```
-1.000000

```

虚数解の場合
```
0.500000+0.866025i
0.500000-0.866025i

```


### 注意事項
aの値は0ではないという前提でプログラムを作成して構いません(必ず2次方程式であるということがわかっているという前提での課題ということになります)．

必ずプログラムのmain関数より前に"#include <math.h>"と記載し，mathライブラリをincludeしてください．
平方根計算の際に利用すると思います．

出力は最後の値の後に必ず改行を行ってください．
改行を行わないとテストに通りません．
