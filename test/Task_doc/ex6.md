# 第6回課題

## 課題1(11-Sorting.pdfの課題22-1)
11-Sorting.pdfを参考に入力された配列をバブルソートで整列するプログラムを作成しなさい．
バブルソートは関数"bubble"の中で実装すること．

### 入力形式
```
(ソートする配列の要素数N)
(値1)
(値2)
・・・
(値N)
```
入力は全て整数で与えられます．

### 出力形式

```math
(1番小さい値)
(2番小さい値)
・・・
(N番目に小さい値)
```

出力は全て整数で行ってください．

### 具体例
*入力例1*
```
10
8
6
2
1
2
4
2
8
9
5
```

<div style="page-break-before:always"></div>

*出力例1*
```
1
2
2
2
4
5
6
8
8
9
```

### 注意事項
特になし．

```
(値1)
(値2)
・・・
(値N)
```

入力は全て整数で与えられます．

### 出力形式
```math
(1番小さい値)
(2番小さい値)
・・・
(N番目に小さい値)
```

出力は全て整数で行ってください．

### 具体例
具体例は課題1と同様です．

### 注意事項
与えられるテスト形式はバブルソートと同じです．
しかし，実行時間に制約を設けており，バブルソートでは制限時間以内に実行が終了せずにエラーとなります．
実行時間のエラー時には以下のメッセージが出ます．
```
***Time limit exceeded***
```
