### 入力形式
```
(入力データ数)
(値1)
(値2)
・・・
(値n)
```
入力データ数は自然数で与えてください．
そのほかの入力は**小数**でも受け付けられるようにしてください．
浮動小数点の範囲("float"型の範囲)で受け付けられるようにしてください．

### 出力形式
```
(平均の値)
(分散の値)
```

出力桁数は小数第4位を四捨五入して，**小数第3位**までで行ってください．
