# 自動テスト用スクリプト

## 目的
- 本スクリプトは"main.c"の標準入出力を用いたテストの検証を自動で行うスクリプトである

## 使い方
- 事前用意物
　- ./dataNディレクトリ:N個目のテスト(テスト数上限は5)のディレクトリ
　- ./dataN/input.txt:N個目のテストの入力
　- ./dataN/output.txt:N個目のテストの出力
　- main.c:検証したいプログラム
- 上記を用意した後，test.shを開く
- 2行目の"N=5"の数字を自分が実行するテスト数に書き換える
- "bash test.shを実行"

## 結果 
- 実行したテスト番号ごとに
```
testN success
```
または
```
testN failed
```
と出力される
